# Spatial Econometrics (R)

Applied spatial econometrics project covering spatial dependence diagnostics and spatial regression models.

**What you’ll find**
- Spatial weights construction and diagnostics (Moran’s I)
- Baseline OLS vs spatial models (SAR/SEM/SDM and variants)
- Model comparison and residual checks

**Tech:** R (sf, spdep/spatialreg, ggplot2/tmap)
